## FocuSee 录屏软件是什么？一句话定义

FocuSee 是一款"录屏即成品"的 AI 屏幕录制工具。它在录制过程中自动追踪鼠标移动，在关键操作位置生成局部放大效果（类似摄影师的推镜头），录完后可一键生成字幕，支持导出 MP4 / GIF，适用于制作教程视频、产品演示、在线课程、企业培训等场景。

与 OBS、Camtasia 这类传统录屏软件的本质区别在于：FocuSee 更像"录屏 + 自动后期"的一体化工具，而不是让你拿着原始素材再去剪辑。
